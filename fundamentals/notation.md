# Chess Notation (Algebraic)

Standard Algebraic Notation (SAN) is the universally used system for recording chess moves.

---

## The Board

Each square has a unique name: **file letter** (a–h, left to right from White's view) + **rank number** (1–8, bottom to top from White's view).

```
  a   b   c   d   e   f   g   h
8 [r] [n] [b] [q] [k] [b] [n] [r]  ← Black's back rank
7 [p] [p] [p] [p] [p] [p] [p] [p]
6 [ ] [ ] [ ] [ ] [ ] [ ] [ ] [ ]
5 [ ] [ ] [ ] [ ] [ ] [ ] [ ] [ ]
4 [ ] [ ] [ ] [ ] [ ] [ ] [ ] [ ]
3 [ ] [ ] [ ] [ ] [ ] [ ] [ ] [ ]
2 [P] [P] [P] [P] [P] [P] [P] [P]
1 [R] [N] [B] [Q] [K] [B] [N] [R]  ← White's back rank
```

## Writing Moves

| Move Type | Format | Example |
|-----------|--------|---------|
| Piece move | Piece + destination | Nf3, Bb5, Qd2 |
| Pawn move | Destination only | e4, d5 |
| Capture | Insert "x" | Bxe5, Nxd4, exd5 |
| Disambiguation | Add file/rank of origin | Rae1, N6d5, Raxb1 |
| Castling | O-O (kingside), O-O-O (queenside) | O-O |
| Check | + | Bb5+ |
| Checkmate | # | Qh7# |
| Promotion | =Piece | e8=Q |
| En passant | Normal capture notation | exd6 |

---

## Game Results

- **1-0** — White wins
- **0-1** — Black wins
- **½-½** — Draw
