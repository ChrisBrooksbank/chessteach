# Chess Teaching Wiki

A comprehensive, interlinked knowledge base for learning and teaching chess — covering openings, tactics, middlegame strategy, endgames, and fundamentals.

## Structure

| Section | Description |
|---------|-------------|
| [Openings](openings/index.md) | Complete guide to chess openings at every level |
| [Tactics](tactics/index.md) | Tactical motifs and combinational patterns |
| [Middlegame](middlegame/index.md) | Positional strategy and planning |
| [Endgames](endgames/index.md) | Endgame theory and technique |
| [Fundamentals](fundamentals/index.md) | Core chess principles for all levels |
| [Famous Games](famous-games/index.md) | Landmark games every player should study |

## How to Use This Wiki

- **Beginners**: Start with [Fundamentals](fundamentals/index.md), then explore [Tactics](tactics/index.md)
- **Intermediate**: Study [Openings](openings/index.md) alongside [Middlegame Strategy](middlegame/index.md)
- **Advanced**: Deep-dive into specific [Endgame](endgames/index.md) theory and complex [Tactical Motifs](tactics/index.md)

Every page cross-links to related concepts. Follow the links to build a connected understanding of how chess ideas relate to each other.

## Navigation

- Each section has its own `index.md` with a table of contents
- Internal links connect related concepts across sections
- Use your editor's or GitHub's search to find specific topics
