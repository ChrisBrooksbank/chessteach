# CLAUDE.md

## Project Overview

A comprehensive Markdown wiki for learning chess — covering openings, tactics, middlegame strategy, endgames, fundamentals, and famous games. Structured as a progressive learning path with cross-linked concepts.

## Repository Structure

This is a documentation repository — pure Markdown files, no build step.

```
openings/     — Opening theory and variations
tactics/      — Tactical motifs and combinations
middlegame/   — Positional strategy and planning
endgames/     — Endgame theory and technique
fundamentals/ — Core principles for all levels
famous-games/ — Landmark games analysis
```

Each section has an `index.md` table of contents.

## Content Guidelines

- Cross-link related concepts between sections
- Include FEN/PGN notation for positions and games where relevant
- Structure content at multiple skill levels (beginner → intermediate → advanced)
- Keep `README.md` section overview table up to date

## No Build Required

This is a static Markdown wiki. Edit `.md` files directly.
